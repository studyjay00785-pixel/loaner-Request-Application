# Loaner Request Application

> A ServiceNow scoped application for managing and tracking loaner requests for laptops, mobile phones, and projectors.

---

## Business Problem

Loaner Requests for laptops, mobile phones, and projectors have historically been managed through a mix of **email**, **spreadsheets**, and **personal contact**. This fragmented approach has created several pain points:

- Requests are difficult to **track** and **report on**.
- Some requests have gone **unfilled** entirely.
- The mismanagement is **dissatisfying to employees** and **expensive for the company**.

The Loaner Request application will manage and track Loaner Requests through an **easy-to-use user interface** that is consistent with other corporate software.

---

## Desired Outcomes

| # | Outcome |
|---|---------|
| 1 | A **single location** where employees can submit Loaner Requests for laptops, mobile phones, and projectors. |
| 2 | **Accurate and timely completion** of all tasks necessary to fulfill Loaner Requests; fewer missed deadlines. |
| 3 | **Retire the spreadsheets** currently used to manage Loaner Requests. |
| 4 | **Improve requestor satisfaction** with the process and request fulfillment. |
| 5 | **Automation** of all steps possible. |

---

## Scope

The application will support loaner management for the following asset types:

- 💻 **Laptops**
- 📱 **Mobile Phones**
- 📽️ **Projectors**

---

## Key Benefits

- ✅ **Centralized request management** — one place for all loaner requests
- ✅ **Improved visibility** — real-time tracking and reporting
- ✅ **Automated workflows** — reduce manual effort and missed steps
- ✅ **Better employee experience** — consistent, easy-to-use interface
- ✅ **Cost reduction** — eliminate inefficiencies of email/spreadsheet-based management

---

## Technology

Built on the **ServiceNow** platform, leveraging:

- Custom tables and forms
- Workflow / Flow Designer for automation
- Service Catalog for end-user request submission
- Reporting and dashboards for management visibility
